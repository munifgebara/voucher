REQUISITOS VOUCHER

ALUNO

  • Aluno gera uma lista de vouchers
  • Cada voucher pode ser utilizado apenas uma vez
  • Recebe 25% de desconto na mensalidade para cada voucher utilizado por um Candidato, caso o candidato pague a primeira mensalidade
  • Aluno que também é colaborador não pode gerar vouchers

CANDIDATO

  • Quem realiza uma inscrição
  • Utiliza opcionalmente um voucher
  • Realiza o pagamento do boleto de primeira mensalidade
  • Candidato que também é colaborador não pode gerar vouchers

VOUCHER

  • Pertence a uma campanha
  • Só pode ser utilizado nos concursos liberado, caso tenha vínculo com N concursos
